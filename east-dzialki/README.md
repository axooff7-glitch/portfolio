# east-dzialki 🏡

Zaawansowany system działek 50x50, charakteryzujący się unikalną mechaniką tworzenia opartą na craftingu oraz wysokiej jakości interfejsem graficznym.

## ✨ Funkcje

- **🔨 Tworzenie przez Crafting**: Działki tworzy się poprzez postawienie specjalnego "Bloku Działki" (Ognisko Dusz), stworzonego według niestandardowej receptury 3x3.
- **📐 Stałe Wymiary**: Standaryzowane działki 50x50 dla łatwiejszego zarządzania terenem.
- **🛡️ Ochrona Serca**: Środek działki ("Serce") jest całkowicie niezniszczalny (odporny na wybuchy, tłoki i niszczenie).
- **📊 Limity Rang**: Konfigurowalne limity działek dla różnych rang (np. VIP, SVIP) poprzez uprawnienia.
- **💡 Informacje Wizualne**:
  - Hologramy nad sercem działki z nazwą właściciela.
  - Własny BossBar pokazujący właściciela terenu po wejściu na działkę.
  - Efekty Blindness i Title (odliczanie) podczas teleportacji.
- **🖥️ Panel Premium**: Profesjonalne menu zarządzania z wyśrodkowanym tytułem ᴘᴀɴᴇʟ ᴅᴢɪᴀʟᴋɪ i ozdobną stylistyką.
- **📆 System Ważności**: Działki posiadają datę wygaśnięcia. Przedłużenie odbywa się za pomocą Bloków Diamentu.

## 🛠️ Komendy

| Komenda | Aliasty | Opis |
| :--- | :--- | :--- |
| `/dzialka dom` | `/p home` | Teleportacja na działkę z 5-sekundowym odliczaniem. |
| `/dzialka panel` | - | Otwiera GUI zarządzania (członkowie, ważność, status). |
| `/dzialka usun` | - | Inicjuje usuwanie działki z potwierdzeniem w GUI. |
| `/dzialka lista` | - | Wyświetla listę wszystkich posiadanych działek. |
